# Interface - PRIME COFFEE

Landing page desenvolvida como parte do **Trabalho Final da disciplina de Interface Homem-Máquina (IHM)** do **IF Baiano – Campus Guanambi**.  
O projeto busca representar visualmente a marca **PRIME COFFEE**, destacando sua identidade sofisticada e aconchegante, com foco em **usabilidade, estética e responsividade**.

---

## Objetivo

O objetivo do projeto é **transformar o protótipo do Figma em uma interface web real**, utilizando **HTML e CSS puros**, respeitando princípios de:
- Design centrado no usuário;
- Hierarquia visual e tipográfica;
- Harmonia de cores e contraste;
- Responsividade para diferentes tamanhos de tela.

---

## Protótipo no Figma

Acesse o design original:  
[Figma - Interface PRIME COFFEE](https://www.figma.com/design/pPTfAGva457DOv4BGHqaTC/Interface---Cafeteria?node-id=0-1)

O protótipo foi criado no **Figma**, com uma identidade visual que reflete o conceito premium da marca:  
- Paleta de cores quentes (tons de marrom, dourado e bege);  
- Tipografia **Poppins**;  
- Layout limpo e acolhedor, inspirado em cafeterias artesanais.

---

## Tecnologias utilizadas

| Tecnologia | Descrição |
|-------------|------------|
| **HTML5** | Estrutura semântica da página |
| **CSS3** | Estilização e responsividade |
| **JavaScript** | Funcionalidades interativas, como linkagem de botões, navegação e ações dinâmicas |
| **Google Fonts** | Tipografia “Poppins” |
| **Git/GitHub** | Controle de versão e hospedagem do projeto |
